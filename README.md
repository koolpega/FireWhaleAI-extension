# 📦 Installation

## Chrome / Chromium Browsers

### 1. Clone the Repository

```bash
git clone https://github.com/koolpega/FireWhaleAI-extension.git
```

### 2. Open Extensions Page

Go to:

```text
chrome://extensions
```

### 3. Enable Developer Mode

Turn on **Developer Mode** from the top-right corner.

### 4. Load the Extension

Click:

```text
Load unpacked
```

Then select the project folder.

# ⚙️ Configuration

Add your API keys and configuration values securely.

Example:

```env
GEMINI_API_KEY=your_api_key_here
```

Get your Gemini API Key from here: https://aistudio.google.com/api-keys

---

# 🔐 Security Recommendations

To keep your API keys secure:

- Use environment variables
- Avoid hardcoding secrets in frontend files
- Use backend proxy endpoints
- Add sensitive files to `.gitignore`
